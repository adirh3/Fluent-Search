## Calculator Search App in Fluent Search

The **Calculator** Search App lets you do quick calculations directly from Fluent Search.

It’s ideal for “lightweight math” while you’re working: conversions, quick estimates, percentages, and basic arithmetic.

---

### Common use cases

- `15% of 2499`
- `1024 * 768`
- `12.5 / 3`

---

### Tips

- If you don’t want calculations to appear in “search anything”, set Calculator to **Search Tag Only** and use the `Calculator` tag when you need it.
