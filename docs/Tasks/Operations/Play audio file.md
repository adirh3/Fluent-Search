# Play audio file

**Type:** `Play audio file` | **Category:** Sound | **Icon:** ðŸ”Š

Plays a sound from an audio file. The sound cannot be paused once started.

---

## Settings

| Setting | Type | Default | Description |
|---|---|---|---|
| **File Path** | string | `""` | Path to the audio file. Supports variable substitution |
| **Cache Audio** | bool | false | Cache the audio stream in memory for faster subsequent playback |

---

## Outputs

None.
