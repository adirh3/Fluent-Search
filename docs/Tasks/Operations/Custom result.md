# Custom result

**Type:** `Custom result`

Creates a result row inside Fluent Search.

## Common settings

- **name**: Text shown in the UI; often templated (for example `Go to {searchText} in Discord`).
- **type / group / score**: Used for grouping and ordering.
- **previewImageProviderSetting / iconGlyph**: Visuals.
