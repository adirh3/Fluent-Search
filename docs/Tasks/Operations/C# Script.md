# C# Script

**Type:** `C# Script`

Runs a C# snippet (often to transform text, parse regex matches, or build custom UI elements).

## Common settings

- **enableVariableInline**: Allows inline variable substitutions.
- **content**: C# content that returns a value.

## Tips

- Use C# scripts for fast text transformations; use HTTP/PowerShell for external effects.
- Keep scripts small and deterministic.
